# Excel Dashboard: Global Suicide Rates Analysis

## Project Overview
This project presents an interactive Excel dashboard analyzing global suicide statistics to uncover trends by age group, gender, generation, and country.

The project focuses on Excel-based data analysis and visualization without coding.
